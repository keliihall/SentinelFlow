## Residual Risks

- Development authentication is intentionally simple and must be replaced for any
  production-like pilot or protected by a trusted reverse proxy.
- Compose deployment is single-node and local-volume based.
- PostgreSQL is documented as reserved/not active for v1.0-rc; SQLite is the
  supported deployment backend.
- Performance baseline is scoped to single-node local pilots and synthetic
  example workloads; it is not evidence for distributed production scale.
- Docker image build time depends on compiling the Rust workspace inside the image.

## Current Limits And Next Stage

Not included in P5.5/v1.0-rc:

- Plugin marketplace.
- Distributed Worker.
- AI automatic analysis.
- Advanced team spaces.
- Real scanner, exploit, brute-force, stealth probing, persistence, credential
  bypass, or attack-chain automation.

Next-stage P6 planning can cover plugin ecosystem, team collaboration,
distributed Worker execution, and AI-assisted analysis over already-normalized
safe findings, but those capabilities are not implemented in this candidate.
