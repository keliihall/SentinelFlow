# Planning Archive

This directory contains non-normative planning and task-breakdown material for
SentinelFlow.

Use the delivery-oriented documents in the repository root and main `docs/`
areas for current user guidance, release gates, deployment, protocol, security
boundaries, troubleshooting, and plugin development.

Planning documents here may describe ideas, sequencing, or historical product
thinking. They do not override the current v1.0-rc scope:

- SentinelFlow is a network security validation tool management framework.
- Safe examples and tests use local synthetic data only.
- Real scanners, exploits, brute-force, stealth probing, persistence,
  authentication bypass, and attack-chain automation are not included.
